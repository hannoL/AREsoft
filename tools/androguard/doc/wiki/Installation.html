Installation

Download

You can the latest development version : 

hg clone https://androguard.googlecode.com/hg/ androguard

Requirements

  • >= python 2.6
  • networkx please install the mercurial version
  • ipython for androlyze
  • python-ptrace for androdump.py
  • pydot for androdd.py or method2 methods.
  • chilkat can be used to unzip APK application, otherwise the zip python module is used (module apk.py)
  • magic is used in method get_files_types in APK module to found files types (module apk.py)
  • pyfuzzy is used to calculate risk indicator (module risk.py)
  • psyco is used to accelerate androguard, but it's not mandatory to install it.

For now, androsim.py + androdiff.py + androsign.py + androcsign.py are only available for linux 32 bits. But you can compile librairies to use them. But you must have the following libraries on your system :

  • bzip2
  • zlib
  • xz
  • snappy
  • sparsehash
  • python-dev

You can compile librairies direcly with the Makefile in the root directory : 

desnos@destiny:~/androguard$ make

Run tools

After the installation of requirements, you can direcly run the tools in the main directory of androguard.

Examples : 

$ ./demos/dalvikvm_format_1.py
$ ./androlyze.py -i ~/Téléchargements/com.rovio.angrybirdsseasons-1.apk -m . -p

So, you have few demos in the demos directory, and all tests are in tests directory.
